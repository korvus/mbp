# Qu'est ce que c'est

Juste une carte avec les pâtisseries de Paris rencontrant un certain succés. Le classement est totalement subjectif, essentiellement basé sur des commentaires receuillit sur des groupes facebook. 
[https://patisseries.vercel.app/](https://patisseries.vercel.app/)

# Pourquoi ce projet

Dans le cadre d'une réorientation professionnelle vers le milieu de la boulangerie, j'ai créé une carte des meilleurs boulangerie (https://meilleuresbaguetteparis.vercel.app/) de Paris, et celle sur les patisseries n'est qu'un fork celle ci.
